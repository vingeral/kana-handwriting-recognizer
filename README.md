# Kana Handwriting Recognizer

An iOS handwriting recognition demo for Japanese kana using CoreML + PencilKit.

## Features
- Handwriting input with PencilKit
- CoreML image classification
- Top-10 prediction display
- White background + black ink normalization
- Supports Hiragana & Katakana

## Requirements
- iOS 17+
- Xcode 15+
- CoreML model trained with Create ML

## How to run
1. Open `KanaRecognize.xcodeproj`
2. Select a simulator or device
3. Build & Run

## Model
The CoreML model file is included / or needs to be trained separately.
